Oliver’s Market

Sistema de caixa (PDV) desenvolvido com HTML, CSS e JavaScript puro, focado em simular o fluxo real de uma compra em mercado.

Sobre o projeto

O Oliver’s Market é um sistema simples de ponto de venda que permite:

Adicionar produtos ao carrinho
Visualizar o total da compra
Aplicar desconto
Inserir valor pago
Calcular troco automaticamente
Finalizar a compra com confirmação visual

O projeto foi desenvolvido com foco em lógica, organização e experiência do usuário (UX).

Funcionalidades
🛒 Adição de produtos ao carrinho
💰 Cálculo automático do valor total
🧮 Aplicação de desconto
💵 Cálculo de troco
🧾 Tela de resumo da compra
✅ Tela final de confirmação (compra concluída)
🔄 Reset automático do sistema após finalização
🚧 Botões preparados para futuras funcionalidades
